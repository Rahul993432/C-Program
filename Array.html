ARRAY :-

Array is a linear data structure, data can store multiple value at a time.
In C array is a variable that can store multiple value at a time.

Array Declaration :-

syntax :-
dataType arrayName[size];
dataType -> any valid C dataType
arrayName -> any valid C identifier
size -> Any non zero positive integer literal.

Ex :-
int number[10]; //here numbers is an integer type array, it can store 10 different integer numbers at a time.
float marks[20]; //here marks is a float array,it can store 20 different float numbers at a time.
int n[100]; //here n is an integer array of capacity 100. That means it can store 100 different integer numbers at a time.

How much memory is a allocated for an array ? :-

Total memory size allocated by an array = size of arrayType * size of array
Ex :-
int x[20]; //here: x is an array of capacity 20 of integer type. Memory size allocated for x=size of int * size of array. 4 bytes*20 = 80 Bytes.
char c[50]; //here: c is an array of capacity 50 of char type. Memory size allocated for c=size of char * size of array. 1 bytes*50 = 50 bytes.

For an array always contigeous memory is allocated.
In an array every item is known as ELEMENT. Every element is identified by a number known as IndexNumber.

Ex :-
int arr[5];
for arr, memory will be allocated as:
Name:arrsize:20bytes In C array index always begins from 0(zero).

Accessing element of an array :-

syntax :-
arrayName[indexNumber];
Index number must be within LOWER BOND (0) and UPPER BOND (arraySize-1).
arraySize=10
=10-1 => 9
Note :-We can use only one element of array at a time.

Traversal of an array :-

The process of accessing every element of an array once.
To traverse an array a loop is used. Generally 'for' loop is used to traverse the array because the size of an array is fixed and the loop counter is used as index number of array as:
for( i=0 ; i < sizeOfArray ; i+=1)
{
process ith element of array;
}
Forward traversal - from begning to end of array.
Backward traversal - from end to begning of array.
backward Traversal :-
for( i=sizeOfArray-1 ; i >=0 ; i-=1)
{
process ith element of array
}

Swapping :-

Process of exchanging value of two variable.

Swapping using third container:-

int x,y,z;
x=10;
y=20;
swap x and y
  1. Put the value of any variable in temprory variable.
    2. z=x;
  2. Now put the value of another variable in variable which value is stored in temprory variable.
    3. x=y;
  3. Now put the value of temprory variable in another variable.
    4. y=z;
z=x;
x=y;
y=z;
can swap any types of variables.

Swapping of integer variable. (Without using third variable concept) :-

int a,b;
a=10;
b=20;
swap a and b
  1. Add value of a and b and put sum in a.
    a=a+b; a will be 10+20=30 (new a=30 and b=20)
  2. Subtract value of b from a and put result in b.
    b=a-b; b will be 30-20=10 (new a=30 and b=10)
  3. again subtract value of b from a and put value in a.
    a=a-b; a will be 30-10=20 (now a=20 and b=10)
a=a+b;
b=a-b;
a=a-b;
will be valid only if sum of both variable is not more than the range of variable.
