Function/Procedure/Sub-routine/Method :-

It is a code block written for specific task. And called whenever required. Every function has unique name by which function/code block is called.

Some function takes some value to work, it is known as argument.
Some function return value after performing operation, it is known as return value.

There are two types of function according to defination :-

  1. Function declaration (Only once)
  2. Function defination (Only once) (Not in predefined function)
  3. Function calling (Infinite times)
  1. Function declaration :-

    2. Syntax :-
    returnType functionName(argument1Type,argument2Type,argumentNType);
    returnType :- Any valid C data type
    argument1Type,argument2Type :- Any valid C data type
    functionName :- Any valid C identifier

    ex :-
    int sum(int,int);
    It declares 'sum' is a function that will accept two integer number and after performing operation it will return an integer number.

    float kush(int,float,float);
    It declares 'kush' is a function that will accept one integer and two float number and after performing operation it will return a float number.

    Note :- A function can accept many arguments but can return only one value.

  2. Function defination :-

    3. Syntax :-
    returnType functionName(argumentType varName,argumentType varName) <-Function Header
    {
    //Local variable declaration
    statement(s);
    return valueTobeReturn;
    }     <-All are Function body

    ex :-
    Creating a function that will accept two integer number and return the sum of both numbers the sum of two integer number will be an integer number so, that return type of function will be int.

    //declaration
    int sum(int,int);

    //defination
    int sum(int a,int b)
    {
    int c;
    c=c+b;
    return c;
    }

  3. Function call :-

    4. Syntax :-
    It function return value and want so store returned value:
    varName = functionName(argumentList);

    If function dosn't return value:
    functionName(argumentList);
